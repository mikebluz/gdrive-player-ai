Serialbox

This player will read a file in the root of your Google Drive called Serialbox Playlists. In this file, list the folder names which you want the player to treat as playlists, i.e., contain audio files. You can also create a Google Doc in that folder called Artist name containing the name of the artist and it will populate in the player. You can also add an image to the folder called Artwork and it will be used as the artwork.
